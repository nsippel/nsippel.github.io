About .1‑to.com.

I wanted to build website for my portfolio and I decided to host it at Github and using Jekyll. Jekyll is written in Ruby. It is easy and awesome. You need to have Ruby and Bundler installed on your local machine, write four command lines, browse to http://localhost:4000 and you will see your new site. I tweaked the color scheme and used Bootstrap Framework to customize the site.

 

~ $ gem install jekyll # 1: Install jekyll in your local machine.

~ $ jekyll new 1-to # 2: This will create a new folder 1-to and generate a new jekyll site in the new folder.

~ $ cd 1-to # 3: Don't forget to go to the site folder.

~ $ jekyll serve # 4: serve jekyll.

# Browse and find the new site at the given Server Address http://localhost:4000
